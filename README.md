# calc_assignment

This calculator was developed as part of a mobile application development assignment using a **ValueNotifier** architecture with dashboard and tab navigation for state management. The application supports all required basic arithmetic operations, including addition, subtraction, multiplication, division, percentage calculations, sign toggle, and backspace/clear functionality. It uses a **midnight_blue** theme and follows a simple project structure with separate screens and controllers.

In addition to the assignment requirements, the calculator includes several bonus scientific features such as trigonometric and hyperbolic functions, logarithms, power operations, factorial, permutations (nPr), combinations (nCr), and mathematical constants like π and e. To run the project, simply execute `flutter pub get` followed by `flutter run`.
